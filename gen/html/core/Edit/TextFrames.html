TextFrames Module¶

About¶

Command Guide¶

Module TextFrames implements the operations on text frames in Oberon.

This is the heart of Oberon’s text-based user interface.

API Guide¶

Dependencies¶

Modules Input Display Viewers Fonts Texts Oberon MenuViewers

Line = POINTER TO LineDesc;
   LineDesc = RECORD
     len: LONGINT;
     wid: INTEGER;
     eot: BOOLEAN;
     next: Line
   END;

   Location* = RECORD
     org*, pos*: LONGINT;
     dx*, x*, y*: INTEGER;
     lin: Line
   END;

   Frame* = POINTER TO FrameDesc;
   FrameDesc* = RECORD
     (Display.FrameDesc)
     text*: Texts.Text;
     org*: LONGINT;
     col*: INTEGER;
     lsp*: INTEGER;
     left*, right*, top*, bot*: INTEGER;
     markH*: INTEGER;
     time*: LONGINT;
     hasCar*, hasSel*, hasMark, atEnd: BOOLEAN;
     carloc*: Location;
     selbeg*, selend*: Location;
     trailer: Line
   END;

   UpdateMsg* = RECORD (Display.FrameMsg)
     id*: INTEGER;
     text*: Texts.Text;
     beg*, end*: LONGINT
   END;

   CallMsg* = RECORD (Display.FrameMsg)
     offset*: INTEGER
   END;

   CopyOverMsg = RECORD (Display.FrameMsg)
     text: Texts.Text;
     beg, end: LONGINT
   END;
