OXLinker Module¶

About¶

Command Guide¶

Module OXLinker transforms a linkable binary module into a standalone binary suitable for installation in the boot area of an Oberon filesystem.

``` OXP.Compile Modules.Mod OXLinker.Link Modules OXLinker.Load Modules

OXP.Compile BareMetalR.Mod OXLinker.Strip BareMetalR ```
