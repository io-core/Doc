HAL.a64.pbpro Module¶

About¶

Command Guide¶

Module HAL (.a64) exercises features of the compiler in bringing Oberon up from bare metal on aarch64 on the Pinebook Pro.

Development Guide¶

The HAL prepares the Oberon runtime and so cannot rely on it.

  • No global variables

  • No strings

  • No heap allocation

The first thing HAL must do is set up its own stack.

Each platform should have its own HAL. This is the HAL for aarch64 on the Pinebook Pro.

The HAL remains resident and may be used by other modules.
