Oberon implements the user-interaction functionality and the task loop of the system. 

.. list-table::  
   :widths: 20 80 
   :header-rows: 1 

   * - Module 
     - Description 

   * - Oberon 
     - implements the Oberon Textual User Interface. 
   * - Viewers 
     - partitions and manages display panes. 
   * - Display 
     - renders text and graphics to the framebuffer. 
   * - Input 
     - presents keyboard and mouse events to the Oberon UI
